# Notification Sounds

This directory contains notification sound files for PWA notifications.

## Required File

**notification.mp3** (or **notification.ogg**)
- Short notification sound/ringtone (1-3 seconds)
- Optimized for mobile (under 50KB recommended)
- Common formats: MP3, OGG, WAV

## Usage

The notification system will automatically play this sound when:
- Push notifications are received
- In-app notifications are shown
- Urgent task/reminder notifications arrive

## Adding a Sound File

1. Download or create a short notification sound
2. Optimize for web (compress, keep under 50KB)
3. Place in this directory as `notification.mp3` or `notification.ogg`
4. The code will automatically detect and use it

## Example Sources

- Free notification sounds: freesound.org, zapsplat.com
- Generate: Use online audio tools to create a short beep/chime
- Custom: Record or create your own notification sound
